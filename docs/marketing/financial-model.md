# 💰 Financial Model — Nexphoria Peptide Brand
> Lean startup model | Updated June 2026
> **Assumptions:** Avg order = $120 | COGS = $48 (40%) | Cold-chain shipping = $25 (charged to customer, cost-neutral) | Net margin per order = $72

---

## 📌 Key Assumptions

| Metric | Value | Notes |
|--------|-------|-------|
| Average Order Value (AOV) | $120 | 1–2 peptides per order |
| COGS per order | $48 | 40% of AOV (product + packaging) |
| Cold-chain shipping | $25 | Charged to customer = cost-neutral |
| Gross profit per order | **$72** | $120 - $48 = $72 |
| Gross margin | **60%** | Before opex |
| Monthly fixed costs (lean) | ~$2,000–$3,000 | Platform, hosting, email tools, LLC |
| Customer repeat rate (est.) | 35–45% | Peptide users reorder every 4–6 weeks |
| Chargeback/refund rate | 2–3% | Budgeted into COGS |

## 🚀 Path to $100K MRR

| Metric | Target |
|--------|--------|
| Orders required | **834 orders/month** at $120 AOV |
| OR with better AOV | **555 orders/month** at $180 AOV (bundles + upsells) |
| Timeline (Scenario C) | ~24–30 months post-launch |
| Key lever: AOV growth | Push bundles, subscriptions, add-ons to reach $180+ AOV |
| Key lever: LTV | 35–45% repeat rate → strong email flows needed |

### 📈 AOV Uplift Strategy
- **Bundle discount:** BPC-157 + TB-500 + BAC water = $185 (vs $140 separate)
- **Subscribe & Save:** 15% off monthly auto-ship
- **Starter kit:** Peptide + syringes + storage bag = $200+ AOV
- **Stack guide upsell:** Add $20–$30 protocol guide at checkout

| Ad platform bans | High | Rely on SEO + community, not Meta/Google ads |
